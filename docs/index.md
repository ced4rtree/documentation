# Tator Tome

Welcome to the Tator Tome! This site is home to the documentation that Team
2122 has compiled over the years to give students and mentors alike a reference
point to maintain consistent, quality codebases, as well as document various
processes that may be useful.

Other useful documentation sites can be found here:

- [WPILib Documentation](docs.wpilib.org/en/stable)
- [CTRE/Phoenix API Documentation](https://pro.docs.ctr-electronics.com/en/latest/)
- [Rev API Documentation](https://docs.revrobotics.com/brushless)
- [AdvantageKit Documentation](https://docs.advantagekit.org/)
- [Choreo Documentation](https://choreo.autos/) (both app and library)
- [Another website with more documentation links!](https://frc.sh)
